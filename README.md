# Docker Tor

Docker image for [Tor](https://www.torproject.org/).

Currently only used as Tor network proxy daemon allowing connections to port `9050`.


## Samples

### docker-compose

```yaml
version: '3.8'

services:
  app:
    image: xjokay/tor:latest
    ports:
      - 9050:9050
    networks:
      - default
    restart: always
```

### docker run

```sh
docker run -d \
  -p 9050:9050 \
  xjokay/tor:latest
```
